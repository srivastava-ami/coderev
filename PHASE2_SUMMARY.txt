PATTERN EXAMPLES (Quick Reference)
═════════════════════════════════════════════════════════════════════════════

Rule 1: N+1 Queries (BLOCKER)
─────────────────────────────────────────────────────────────────────────────
FIRES (must fix):
  for (const item of items) {
    const user = db.query('SELECT * WHERE id = ?', item.id);  ← N+1
  }

DOESN'T FIRE (OK):
  const users = db.query('SELECT * WHERE id IN (?)', ids);     ← batch load
  items.forEach(item => {
    const user = users.find(u => u.id === item.id);          ← reuse
  });

Rule 2: Memory Allocation (ADVISORY)
─────────────────────────────────────────────────────────────────────────────
FIRES (optimize):
  for (i in 0..n) {
    let buf = Vec::new();              ← allocates per iteration
    buf.push(data[i]);
  }

DOESN'T FIRE (OK):
  let mut buf = Vec::with_capacity(n); ← allocates once with capacity
  for i in 0..n {
    buf.push(data[i]);
  }

Rule 3: Blocking in Async (BLOCKER)
─────────────────────────────────────────────────────────────────────────────
FIRES (must fix):
  async function process() {
    const data = fs.readFileSync(...); ← blocks event loop
    return data;
  }

DOESN'T FIRE (OK):
  async function process() {
    const data = await fs.promises.readFile(...); ← async I/O
    return data;
  }

Rule 4: Unbounded Growth (BLOCKER)
─────────────────────────────────────────────────────────────────────────────
FIRES (must fix):
  const cache = {};
  for (const item of stream) {
    cache[item.key] = item.value;     ← unbounded dict growth
  }

DOESN'T FIRE (OK):
  const cache = {};
  for (const item of stream) {
    if (Object.keys(cache).length >= 10000) break;
    cache[item.key] = item.value;     ← capped (advisory only)
  }


COMMON IMPLEMENTATION PATTERNS
═════════════════════════════════════════════════════════════════════════════

Loop Depth Tracking:
  depth := 0
  for _, ch := range line {
    if ch == '{' { depth++ } else if ch == '}' { depth-- }
  }
  if depth > 0 { inLoop = true }

Async Context Tracking:
  if strings.Contains(line, "async function") ||
     strings.Contains(line, "async () =>") {
    asyncDepth = depth
  }
  if currentDepth <= asyncDepth && currentDepth == 0 {
    inAsync = false  // exited async
  }

Pattern Detection (Language-Agnostic):
  if strings.Contains(line, "db.query") ||
     strings.Contains(line, "db.execute") ||
     strings.Contains(line, "db.find") {
    // potential N+1 query
  }

Size Guard Detection:
  if strings.Contains(line, "length <") ||
     strings.Contains(line, "size <") ||
     strings.Contains(line, ".len() <") {
    // potential size guard; may downgrade to advisory
  }
